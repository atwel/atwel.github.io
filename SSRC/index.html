Eigen and Schuster (1977, 1978)

Eigen and Schuster's answer is to think about species of macro-molecules as being comprised of quasi-species. Here is a sentence they work with, a species if you will. If the reproductive unit is the sentence and reproduction is error prone (all chemical reproduction is) with an equal probability at each letter, the information contained in the sentence would quickly be lost, as degenerate sentences would come to dominate.
But consider treating the words as quasi-species. THe reproducing units are the words and therefore the rates of change (death or growth ) are proportional to current population size. In this schematic, there is no relationship between quasi-species as far as reproduction is concerned.
We can play this out. Here is a column of the original, information rich population. In generation 2, we see two mistakes introduced. Mistakes aren't variable and they die. Eigen and Schuster's insight was that some relational topologies among the quasi-species might help balance reproduction to be error correcting. This cyclic topology can do that under the right conditions>
Here you can see the dynamics of that system, as reported by Eigen and Schuster in there original work.
These dependencies can be written out in the following equations.

$TAKE_{t+1} = \frac{TAKE_t}{ALL} * \frac{ADVANTAGE_t}{ALL} - \frac{TAKE_t}{ALL} * totalgrowth$

$ADVANTAGE_{t+1} = \frac{ADVANTAGE_t}{ALL} * \frac{OF_t}{ALL} - \frac{ADVANTAGE_t}{ALL} * totalgrowth$

$OF_{t+1} = \frac{OF_t}{ALL} * \frac{MISTAKE_t}{ALL} - \frac{OF_t}{ALL} * totalgrowth$

$MISTAKE_{t+1} = \frac{MISTAKE_t}{ALL} * \frac{TAKE_t}{ALL} - \frac{MISTAKE_t}{ALL} * totalgrowth$

Abstracting away from the example and creating a more generic system, we get this differential equation.
$\dot{r}_i = r_i * r_{i+1} - r_i \sum_{j=1}^N r_j * r_{j+1}$
Schematically, we can depict this system like this.
And here are the dynamics of the system for different values of N. Note that this system has time dynamics and selective pressures, but is not full on evolutionary; there is no competition between species because there is a single species compromise of quasi-species. More importantly, notice that the hypercycle is stable only up to N=4. So an improvement, but still not great.
Some years later John Padgett realizes the relevance of this autocatalytic framework for understanding social organization.
To map the insights of the hypercycle model to social production, Padgett and collaborators reframed the system as actors and products in a network like this. The actors are actually skills that can be reproduced by new individuals acquiring them. The skills transform the products along the chain of the network. Importantly, the model is embedded in space, unlike the original hypercycle.

Chemical kinetics says this can be written as a differential or, for our purposes, difference equation.
$Pr(R_{i,t+1} = R_{i,t} + 1) \approx \overbrace{\frac{R_i}{N} * \frac{P_i}{M} * \frac{R_{i+1}}{N}}^\text{growth}$
$\Delta R_i = \overbrace{\frac{R_i}{N} * \frac{P_i}{M} * \frac{R_{i+1}}{N}}^\text{growth} - \overbrace{\frac{R_i}{N} \sum_{j=1}^N \frac{R_j}{N} * \frac{P_j}{M} * \frac{R_{j+1}}{N}}^\text{death}$

$R_{i,t+1} = R_{i,t} + \overbrace{\frac{R_{i,t}}{N} * \frac{P_{i,t}}{M} * \frac{R_{i+1,t}} {N}}^\text{growth} - \overbrace{\frac{R_{i,t}}{N} \sum_{j=1}^N \frac{R_{j,t}}{N} * \frac{P_{j,t}}{M} * \frac{R_{j+1,t}}{N}}^\text{death}$

$R_{i,t+1} = R_{i,t} + \overbrace{r_i p_i r_{i+1}}^\text{growth} - \overbrace{r_i \sum_{j=1}^N r_j p_j r_{j+1}}^\text{death}$

$P_{i,t+1} = P_{i,t} + \overbrace{r_{i-1} p_{i-1} (1-r_i)}^\text{incoming} + \overbrace{r_i p_i (1-r_{i+1})}^\text{outgoing}$
We can describe the non-spatial niche constructing system with the following equation. This derivation is a real thing of beauty, but chemical kinetics says the leading terms dominate the dynamics. Our side by side comparisons show this to be true.
Full dynamics (i.e. with chains)

$R_{i,t+1} = R_{i,t} + \frac{r_i*r_{i+1}}{1-\prod_{j=1}^{N} r_j} * \bigg(p_i + \displaystyle\sum_{k=1}^{N-1}\bigg[p_{i+k}\prod_{h=i+k}^{N-1+i} r_h\bigg]\bigg) - r_i*\bar{f}$

$\bar{f} = \sum r_j$

$P_{i,t} = P_{i,t} + \frac{r_{i+1}(1-r_i)}{1-\prod_{j=1}^{N} r_j}\bigg(p_{i+1} + \displaystyle\sum_{k=i}^{N-2+i}\bigg[p_k \prod_{h=k}^{N-2+i} r_h\bigg]\bigg)\\ - \frac{r_i(1-r_{i+1})}{1-\prod_{j=1}^{N}r_j}\bigg(p_i + \displaystyle\sum_{k=i+1}^{N-1+i}\bigg[p_k\prod_{h=k}^{N-1+i}r_h\bigg]\bigg)$

Leading term dynamics (i.e. no chains)

$R_{i,t+1} = R_{i,t} + r_i p_i r_{i+1} - r_i \sum_{j=1}^N r_j p_j r_{j+1}$

$P_{i,t+1} = P_{i,t} + r_{i-1} p_{i-1} + r_i p_i$
